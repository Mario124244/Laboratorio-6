Inicio Introduccion 1 Complementos
Entity Framework

Un ORM (Object-Relational Mapping) que simplifica el acceso a bases de datos relacionales en aplicaciones .NET.

ASP.NET MVC

Un framework para construir aplicaciones web basadas en el patrón Modelo-Vista-Controlador.

NuGet

Gestor de paquetes para .NET que facilita la incorporación de librerías y componentes de terceros en tus proyectos.

LINQ

Extensión de C# que permite realizar consultas sobre colecciones de objetos, bases de datos y otros orígenes de datos de forma integrada en el lenguaje.

Moq

Biblioteca de pruebas unitarias para crear mocks y simular objetos en pruebas unitarias.

AutoMapper

Biblioteca que facilita el mapeo entre objetos de diferentes clases, útil en aplicaciones que utilizan arquitecturas como DTOs (Data Transfer Objects).

NLog o log4net

Librerías para realizar logging en aplicaciones, permitiendo registrar información de forma estructurada para su análisis posterior.

Dapper

Micro-ORM que permite realizar consultas a bases de datos de forma sencilla y eficiente, con un rendimiento superior en comparación con Entity Framework en ciertos escenarios.

SignalR

Biblioteca para agregar funcionalidades en tiempo real a aplicaciones web, como chat, notificaciones en tiempo real, entre otras.
